## 介绍
### 快速开始
1.环境准备 python3.9

2.安装依赖
``` bash
pip install -r requirements.txt
```
3.运行
```bash
python main.py
```
或
```bash
# 进入web目录 启动django程序
python3 manage.py runserver 0.0.0.0:8000
```
4.访问
初始用户：admin
初始密码：123456

5.升级程序（第一次或者更新都建议执行下）
```bash
# 升级数据库结构 执行库表变更， 老数据不会丢
python update.py
```
